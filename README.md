# ParkingManagementSystem


A Java console application that manages vehicle parking with persistent storage.

## What it does
- Park vehicles and assign them to available spots
- Exit vehicles and calculate parking fees
- Display real-time parking status
- All data is saved to a database and persists after closing

## Technologies used
- Java 22
- SQLite (database)
- JDBC (database connection)
- Maven (project management)

## How to run
1. Clone the repository
2. Open the project in NetBeans
3. Add the SQLite dependency to pom.xml
4. Run the project

## What I learned
- Object oriented programming in Java
- Database integration with JDBC
- SQL injection prevention with PreparedStatement
- Data persistence with SQLite
